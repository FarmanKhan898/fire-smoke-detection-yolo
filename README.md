# Fire & Smoke Detection using YOLO

This project detects fire and smoke in real-time using a YOLO-based object detection model.

## Features
- Real-time detection
- GUI interface
- High accuracy
- No hardware sensors required

## Tech Stack
- Python
- YOLO
- OpenCV
- Tkinter

## How to Run
```bash
pip install -r requirements.txt
python main.py
